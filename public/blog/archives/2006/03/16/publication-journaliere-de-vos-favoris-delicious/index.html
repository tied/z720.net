Publication journalière de vos favoris del.icio.us (archives)

Comme vous l'avez probablement constater dans la rubrique lien, j'ai testé pendant quelques jours un nouveau service del.icio.us : le daily blog posting

Cette option est assez simple : tous les jours del.icio.us se propose de créer un article sur votre blog contenant les favoris que vous avez créés depuis la veille.

Il n'y a pas vraiment besoin de présenter les avantages de ce service, vous n'avez plus besoin de faire un article avec les liens intéressants du jour, on s'en charge pour vous.

Comment ça marche ? Il suffit de s'identifier dans del.icio.us et d'aller dans le menu settings » daily blog posting. Ensuite il vous faut créer un nouveau "job" (add new thingy) auquel vous donnez un nom et vous renseignez :

  • le nom d'utilisateur pour se connecter au blog,
  • et son mot de passe
  • l'adresse du serveur XML-RPC de votre blog
  • l'heure de publication GMT (9 pour 9h GMT soit 10h en France)
  • l'ID du blog pour les systèmes multi-blog. Pour WordPress laissez vide ou mettez 1
  • l'ID de la catégorie dans laquelle le billet sera publié

Pour votre blog WordPress cela devrait donner ceci:

Nom de la zoneValeur d'exemple
job_namePublication Blog
out_nameadmin
out_pass**votre mot de passe**
out_urlhttp://adresse.de/votre/wordpress/xmlrpc.php
out_time8
out_blog_id1
out_cat_idUn numéro de catégorie

Il y a cependant un certains nombre d'inconvénients :

  • Il n'est pas possible de filtrer les liens que vous voulez ajouter à votre article, del.icio.us publie tous vos favoris. La possibilité de filter sur un tag particulier serait un vrai plus.
  • L'article est rédigé en anglais. Il n'est pas possible de paramétrer le titre. C'est un moindre mal dans la mesure où le libellé et le descriptif est saisi par l'utilisateur.
  • Il n'est pas possible de choisir la fréquence : daily, c'est tous les jours. Cependant s'il n'y a pas de nouveaux favoris, aucun article n'est publié.

Conclusion j'ai désactivé cette option pour le moment. Je pense que yadd peut être une bonne base pour un futur plugin sur ce thème. Autre piste à explorer : taga.licio.us
