Extension Wikini (archives)

Publication d'une première version d'une extension de chargement de fichier pour Wikini

Ce week-end, j'ai écrit une petite extension pour Wikini qui permet aux utilisateurs de charger des fichiers dans un répertoire.

Il manque encore pas mal de choses mais il est déjà possible de créer des sous-dossiers et d'y mettre des fichiers. De plus tout le paramétrage est en place.

Mode d'emploi :

  1. Copier le fichier dans le dossier action de votre installation de Wikini
  2. Editer le fichier pour modifier la configuration. (Après les informations de license au début du fichier)
  3. Insérez {{FileUpload}} dans une de vos pages Wikini Attention : toute personne ayant un accès (login) et un droit de lecture sur la page pourra charger des fichiers
  4. C'est tout...
